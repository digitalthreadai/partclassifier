Strategy & Features

PartClassifier

AI-powered mechanical parts classification agent for industrial engineering teams managing large fastener inventories.

Vision

Why PartClassifier

Industrial engineering teams spend hours manually looking up part specifications from distributor websites, extracting dimensions and materials, and preparing classification data for PLM systems. PartClassifier automates this entire workflow -- combining LLM intelligence with multi-source data retrieval to process thousands of parts with minimal human intervention.

The agent bridges the gap between unstructured part catalogs and structured PLM classification systems like Teamcenter. It handles the full pipeline: classification, spec lookup (APIs + web scraping + stealth browser), attribute extraction with unit conversion, and per-class Excel output with canonical column names.

Feature Map

10 integrated features across the classification pipeline, execution modes, and infrastructure.

Core Pipeline 6 features ▼
AI Part Classification src/part_classifier.pyLLM classifies parts into 20+ categories. Configurable provider (Groq, OpenAI, Anthropic, Ollama).
Multi-Source Spec Lookup src/web_scraper.py + src/api_sources.pyPriority: APIs → Stealth Browser → Cache → DuckDuckGo → Stealth Fallback
Attribute Extraction src/attribute_extractor.pyLLM extracts structured key-value attributes with automatic unit conversion (in ↔ mm).
Canonical Normalization src/attr_schema.py120+ alias mappings normalize naming variants to canonical column names per class.
Per-Class Excel Output src/excel_handler.pyGroups parts by type, writes one .xlsx per class with styled headers and auto-sized columns.
URL Caching url_cache.jsonShared across all modes. Cache-hit path skips search. Dead URLs auto-removed.
Execution Modes 3 modes ▼
Streamlit Web UI app.pyInteractive config, file picker, live progress, per-part results, download buttons.
API Key CLI main.pySequential processing. Configure LLM via .env. Best for small batches.
Claude Code CLI main_cc.pyZero API keys. Batch classify 100/call. Parallel workers. Auto-resume. Best for 1K-20K+ parts.
Infrastructure 4 features ▼
Distributor APIs src/api_sources.pyDigiKey (OAuth2), Mouser (API key), McMaster (mTLS). Structured data bypasses LLM extraction.
CloakBrowser Stealth src/stealth_scraper.pyCustom Chromium with 33 C++ patches. Bypasses Cloudflare, reCAPTCHA, FingerprintJS.
curl_cffi TLS Stealth src/web_scraper.pyChrome 124 JA3/JA4 fingerprint impersonation for basic bot detection bypass.
Spec Scoring Algorithm src/web_scraper.pyKeyword density scoring + exact part number bonus. Trusted domains prioritized.
