AI-Powered Classification

PartClassifier

Reads mechanical parts from Excel, classifies each by type, looks up specifications from distributor APIs and web sources, and writes structured Excel output ready for Teamcenter import.

★ GitHub Setup Guide →
Quick Start
$ git clone https://github.com/digitalthreadai/partclassifier.git
$ cd partclassifier && pip install -r requirements.txt
$ cp .env.example .env # configure LLM provider
$ streamlit run app.py # web UI at localhost:8501

Reliability

Bot Detection Bypass

Two layers of stealth ensure maximum site coverage.

Layer 1

curl_cffi

  • Chrome 124 TLS fingerprint (JA3/JA4)
  • Lightweight — no browser download
  • Used for all HTTP calls by default
  • Bypasses basic bot detection that blocks Python's default HTTP stack
Layer 2

CloakBrowser

  • Custom Chromium with 33 C++ source-level patches
  • 0.9 reCAPTCHA v3 score
  • Passes Cloudflare Turnstile, FingerprintJS, 30+ services
  • Full JS rendering for SPAs (McMaster-Carr, etc.)
  • Optional — pip install cloakbrowser (~200 MB)
