Setup Guide

Step-by-step instructions to configure LLM providers, distributor APIs, stealth browser, and run PartClassifier.

1

Python Environment

~3 minutes

Clone the repository and install dependencies in a virtual environment.

Terminal
$ git clone https://github.com/digitalthreadai/partclassifier.git
$ cd partclassifier
$ python -m venv venv
$ venv\Scripts\activate # Windows
$ pip install -r requirements.txt

Installs: openai, openpyxl, curl_cffi, beautifulsoup4, python-dotenv, anthropic, httpx, streamlit, cloakbrowser.

2

LLM Provider

~2 minutes

Copy the example env file and configure your LLM provider.

.env
$ cp .env.example .env
# Edit .env with your chosen provider:
LLM_PROVIDER=groq # groq, openai, anthropic, ollama, custom
LLM_API_KEY=gsk_your_key_here
ProviderDefault ModelGet Key
Groq (free)llama-3.3-70b-versatileconsole.groq.com/keys
OpenAIgpt-4oplatform.openai.com/api-keys
Anthropicclaude-sonnet-4-20250514console.anthropic.com/settings/keys
Ollama (local)llama3.1ollama.ai (no key needed)

Claude Code CLI mode: Skip this step entirely. No .env needed -- uses your claude CLI config.

3

Distributor APIs (Optional)

~5 minutes per API

Higher-quality structured data. When 3+ attributes found via API, LLM extraction is skipped entirely.

.env (optional APIs)
# DigiKey (free: developer.digikey.com)
DIGIKEY_CLIENT_ID=your_client_id
DIGIKEY_CLIENT_SECRET=your_client_secret
# Mouser (free: api.mouser.com)
MOUSER_API_KEY=your_api_key
# McMaster (email eprocurement@mcmaster.com)
MCMASTER_BEARER_TOKEN=your_token
MCMASTER_CLIENT_CERT=path/to/client.pem
MCMASTER_CLIENT_KEY=path/to/client-key.pem
4

Stealth Browser (Optional)

~1 minute + 200MB download

CloakBrowser enables direct scraping of bot-protected sites like McMaster-Carr, Fastenal, and Grainger.

Terminal
$ pip install cloakbrowser
# First run downloads ~200MB Chromium binary automatically
# To disable (keep installed but don't use):
# Add to .env: STEALTH_BROWSER_ENABLED=false

CloakBrowser is a custom-compiled Chromium with 33 C++ source-level patches that bypass Cloudflare, reCAPTCHA v3, and FingerprintJS detection.

5

6

Run the Agent

Depends on part count

Choose your preferred execution mode.

Web UI (recommended)
$ streamlit run app.py
# Opens browser at http://localhost:8501
CLI modes
$ python main.py # API key mode
$ python main_cc.py # Claude Code CLI (zero-key)
$ python main_cc.py --workers 8 # 8 parallel workers

Output files appear in output/, one .xlsx per part class.
