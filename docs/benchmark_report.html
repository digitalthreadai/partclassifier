Key Insights

Opus: Best Attribute Extractor

Opus extracts 4.1 attrs/part on average — 33% more than Sonnet (3.1) and 14% more than Groq (3.6). Wins on 18/44 parts, demonstrating superior ability to parse unstructured web content into structured specs.

Groq: Best Classifier, Rate-Limited

Groq classified 91% of parts vs 70-75% for Claude models. But hit the free-tier limit (100K tokens/day) at ~30 parts, causing 14 parts to get 0 attributes. A paid Groq plan would likely match or exceed Opus.

Web Content Classification Works

The deterministic class extractor (no LLM) classified 70-91% of parts across all models. Saves LLM tokens and ensures consistency. Remaining 10-30% fall back to LLM.

Web Scraping is the Bottleneck

6 parts (14%) got 0 attributes across ALL models. The issue is scraping failures (timeouts, bot blocks), not the LLM. Improving source coverage matters more than switching models.
