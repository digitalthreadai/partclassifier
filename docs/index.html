Pipeline Steps

Six-step pipeline from Excel input to classified output.

1. Read Input

Excel with Part Number, Mfg Part Number, Mfg Name, Unit of Measure

2. Multi-Tier Search

APIs, stealth browser, URL cache, DuckDuckGo, fallback chain

3. Classify

Deterministic from content, LLM cache, or LLM call (60+ classes)

4. Regex Pre-Extract

Tables, key-value patterns, standards, materials via regex

5. LLM Validate

Validates regex, fills gaps, temperature=0 deterministic

6. Write Output

Per-class Excel with canonical column names, styled headers
