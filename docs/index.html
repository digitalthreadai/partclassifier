PartClassifier

Production-grade AI agent for mechanical part classification. Reads Excel input, searches distributor APIs and the web, then runs a classify → blind-extract → validate → extract pipeline across 93 part classes. Schema-driven attribute fitting ensures only valid attributes per class, with per-class Excel output and TC Class ID for Teamcenter PLM import.

Pipeline Steps

Eight-step pipeline from Excel input to classified output with TC Class ID.

1. Multi-Tier Search

7-tier: APIs (DigiKey/Mouser/McMaster) → Stealth Browser → URL Cache → DuckDuckGo → Fallback

2. Classify

Deterministic from web content (95% accuracy) → LLM cache → Batch LLM (100/call)

3. Regex Pre-Extract

Tables, key-value patterns, standards (DIN/ASME/ISO), materials via regex

4. LLM Validate

Hybrid prompt: priority schema hints + maximize coverage, temperature=0

5. Normalize

500+ alias mappings → canonical names, schema-ordered per class

6. Write Output

Per-class Excel with TC Class ID, styled headers, auto-sized columns
